# Admin Skills Management Page

**Location:** `src/app/admin/skills/page.tsx`

## Overview

The Admin Skills Management page allows administrators to create, edit, and delete skills in a hierarchical structure. Skills created here become available for all users to track. Supports two view modes: Tree View (hierarchical) and Grid View (flat cards).

```tsx
<div className="flex p-1.5 bg-gray-100 rounded-2xl">
  <button
    onClick={() => setViewMode("grid")}
```

## Actions

### Create Skill
1. Click "Add Skill" button in header
2. Modal opens with empty form
3. Fill in name (required)
4. Optionally add description
5. Select parent or leave as root
6. Choose color from swatches
7. Search and select icon
8. Click "Create" to save

### Edit Skill
1. Navigate to skill detail page by clicking skill item
2. Click Edit button on detail page
3. Modal opens with current values
4. Modify any field
5. Click "Save" to update

### Delete Skill
1. Click delete action on skill
2. Confirmation modal appears
3. Shows warning if skill has children
4. Click "Delete" to confirm or "Cancel" to abort

### Search Skills
1. Type in search box
2. Results filter in real-time
3. Tree structure preserved (parent paths shown)
4. Grid shows filtered flat list

### Switch View Mode
1. Click "Tree" or "Grid" button
2. View updates immediately
3. Selection preserved across views
