# US-7.3: Fullscreen Mode

## Description

As a **user**,
I want to **view the Pomodoro timer in fullscreen mode**,
So that **I can have a distraction-free, immersive focus experience**.

---

## Acceptance Criteria

- AC-7.3.1: User can enter fullscreen mode via button (top-right corner)
- AC-7.3.2: User can exit fullscreen via button or Escape key
- AC-7.3.3: Fullscreen works on Chrome, Firefox, and Safari (webkit support)
- AC-7.3.4: Button icon changes based on fullscreen state (expand/minimize)
