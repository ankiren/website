# US-7.1: Basic Pomodoro Timer

## Description

As a **user**,
I want to **use a Pomodoro timer with start, pause, reset, and skip controls**,
So that **I can manage my focus sessions using the Pomodoro Technique**.

---

## Acceptance Criteria

- AC-7.1.1: User can start and pause the countdown timer
- AC-7.1.2: User can reset the timer to the full duration
- AC-7.1.3: User can skip to the next session type
- AC-7.1.4: Timer supports 3 session types: Focus (25min), Short Break (5min), Long Break (15min)
- AC-7.1.5: Circular progress indicator shows time remaining visually
- AC-7.1.6: Display count of completed focus sessions
- AC-7.1.7: Play notification sound when timer reaches zero
