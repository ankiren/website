# US-7.4: Custom Duration Settings

## Description

As a **user**,
I want to **customize the duration for each session type**,
So that **I can adapt the Pomodoro timer to my personal productivity preferences**.

---

## Acceptance Criteria

- AC-7.4.1: User can open settings modal via Settings button
- AC-7.4.2: User can adjust duration for Focus (1-60 min), Short Break (1-30 min), Long Break (1-60 min)
- AC-7.4.3: Settings are saved and applied when clicking Save
- AC-7.4.4: User can cancel without saving changes
- AC-7.4.5: Settings persist across browser sessions (localStorage)
