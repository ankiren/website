# US-3.3: Delete Card

**Epic:** [Card Management](EPIC.md)
**Status:** Done
**Priority:** Medium

---

## Description

As a **user**,
I want to **delete cards I no longer need**,
So that **I can keep my decks focused and relevant**.

---

## Acceptance Criteria

- [x] Delete button on each card in deck view
- [x] Confirmation required before deletion
- [x] Deleting card removes all its review history
- [x] Card count updates on deck
- [x] Only card owner can delete

---

## Technical Notes

### Implementation
- **API**: `src/app/api/cards/[id]/route.ts` (DELETE)

### Cascade Delete
```sql
DELETE FROM Review WHERE cardId = ?
DELETE FROM Card WHERE id = ?
```

### Authorization
- Card's deck must be owned by user
- Returns 404 if unauthorized
