# US-3.2: Edit Card

## Description

As a **user**,
I want to **edit my flashcard content**,
So that **I can fix mistakes or improve my cards**.

---

## Acceptance Criteria

- [x] Edit button on each card in deck view
- [x] Edit form pre-filled with current front/back
- [x] Can update front and back content
- [x] Successful update redirects to deck detail
- [x] Only card owner can edit

---

## Technical Notes

### Implementation
- **API**: `src/app/api/cards/[id]/route.ts` (PUT)
- **Page**: `src/app/dashboard/cards/[id]/edit/page.tsx`

### Request/Response
```typescript
// PUT /api/cards/[id]
Request: { front?: string, back?: string }
Response: { id, front, back, updatedAt }
```

### Authorization
- Card's deck must be owned by user
- Returns 404 if unauthorized
