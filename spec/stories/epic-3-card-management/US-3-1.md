## Description

As a **user**,
I want to **add flashcards to my deck**,
So that **I can build my study material**.

## Acceptance Criteria

- [x] "Add Card" button on deck detail page
- [x] Form requires front (question) content
- [x] Form requires back (answer) content
- [x] Card is associated with the correct deck
- [x] Successful creation redirects to deck detail
- [x] New card appears in deck's card list

---

## Technical Notes

### Implementation
- **API**: `src/app/api/cards/route.ts` (POST)
- **Page**: `src/app/dashboard/decks/[id]/cards/new/page.tsx`

### Request/Response
```typescript
// POST /api/cards
Request: { front: string, back: string, deckId: string }
Response: { id, front, back, deckId, createdAt }
```

### Validation
- Front: Required, non-empty
- Back: Required, non-empty
- DeckId: Must be valid deck owned by user

### Authorization
- User must own the deck to add cards
