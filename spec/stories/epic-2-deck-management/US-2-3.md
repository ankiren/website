# US-2.3: View Deck Details

**Epic:** [Deck Management](EPIC.md)
**Status:** Done
**Priority:** High

---

## Description

As a **user**,
I want to **view a deck's details and all its cards**,
So that **I can manage the cards in my deck**.

---

## Acceptance Criteria

- [x] Deck detail page accessible at `/dashboard/decks/[id]`
- [x] Page shows deck name and description
- [x] Page lists all cards in the deck
- [x] Each card shows front and back content
- [x] Each card shows review status (new/due/reviewed)
- [x] Link to add new card
- [x] Link to edit each card
- [x] Link to delete each card
- [x] Link to start study session

---

## Technical Notes

### Implementation
- **API**: `src/app/api/decks/[id]/route.ts` (GET)
- **Page**: `src/app/dashboard/decks/[id]/page.tsx`

### API Response
```typescript
// GET /api/decks/[id]
Response: {
  id: string,
  name: string,
  description: string | null,
  cards: [{
    id: string,
    front: string,
    back: string,
    reviews: [{ dueDate: string }]
  }]
}
```

### Authorization
- Only deck owner can view deck
- Returns 404 if deck not found or unauthorized
