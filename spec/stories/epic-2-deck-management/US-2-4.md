# US-2.4: Edit Deck

## Description

As a **user**,
I want to **edit my deck's name and description**,
So that **I can keep my decks organized as my needs change**.

---

## Acceptance Criteria

- [x] Edit button available on deck detail page
- [x] Edit form pre-filled with current values
- [x] Can update name and description
- [x] Successful update redirects to deck detail
- [x] Only deck owner can edit

---

## Technical Notes

### Implementation
- **API**: `src/app/api/decks/[id]/route.ts` (PUT)

### Request/Response
```typescript
// PUT /api/decks/[id]
Request: { name?: string, description?: string }
Response: { id, name, description, updatedAt }
```

### Authorization
- Validates user owns the deck
- Returns 404 if unauthorized
