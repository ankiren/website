## Description

As a **user**,
I want to **see all my decks on the dashboard**,
So that **I can choose which subject to study**.

## Acceptance Criteria

- [x] Dashboard shows all user's decks
- [x] Each deck card shows name and description
- [x] Each deck shows card count

- [x] Each deck shows new cards count
- [x] Decks link to their detail pages
- [x] Study button links to study session

---

## Technical Notes

### Implementation
- **API**: `src/app/api/decks/route.ts` (GET)
- **Page**: `src/app/dashboard/page.tsx`
- **Component**: `src/components/DeckCard.tsx`

### API Response
```typescript
// GET /api/decks
Response: [{
  id: string,
  name: string,
  description: string | null,
  _count: { cards: number },
  dueCount: number,
  newCount: number
}]
```

### Stats Calculation
- **Total Cards**: COUNT of cards in deck
- **Due Cards**: Cards where dueDate <= today
- **New Cards**: Cards with no Review record
