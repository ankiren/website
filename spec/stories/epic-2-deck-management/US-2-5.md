# US-2.5: Delete Deck

**Epic:** [Deck Management](EPIC.md)
**Status:** Done
**Priority:** Medium

---

## Description

As a **user**,
I want to **delete a deck I no longer need**,
So that **I can keep my dashboard clean and focused**.

---

## Acceptance Criteria

- [x] Delete button available on deck detail page
- [x] Confirmation required before deletion
- [x] Deleting deck removes all its cards
- [x] Deleting deck removes all associated reviews
- [x] Successful deletion redirects to dashboard
- [x] Only deck owner can delete

---

## Technical Notes

### Implementation
- **API**: `src/app/api/decks/[id]/route.ts` (DELETE)

### Cascade Delete
```sql
DELETE FROM Review WHERE cardId IN (SELECT id FROM Card WHERE deckId = ?)
DELETE FROM Card WHERE deckId = ?
DELETE FROM Deck WHERE id = ?
```

### Authorization
- Validates user owns the deck
- Returns 404 if unauthorized
