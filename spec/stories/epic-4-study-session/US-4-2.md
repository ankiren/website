# US-4.2: Flip Card Animation

**Epic:** [Study Session](EPIC.md)
**Status:** Done
**Priority:** High

---

## Description

As a **user**,
I want to **flip the card to reveal the answer**,
So that **I can test my recall before seeing the answer**.

---

## Acceptance Criteria

- [x] Card shows front (question) initially
- [x] Clicking card flips to show back (answer)
- [x] 3D flip animation (rotate on Y-axis)
- [x] Smooth transition (0.6s duration)
- [x] Card maintains perspective for realism
- [x] Rating buttons appear after flip

---

## Technical Notes

### Implementation
- **Component**: `src/components/FlashCard.tsx`

### CSS Animation
```css
.card-flip {
  perspective: 1000px;
}

.card-inner {
  transform-style: preserve-3d;
  transition: transform 0.6s;
}

.card-inner.flipped {
  transform: rotateY(180deg);
}

.card-front, .card-back {
  backface-visibility: hidden;
}

.card-back {
  transform: rotateY(180deg);
}
```

### Component Props
```typescript
interface FlashCardProps {
  front: string;
  back: string;
  isFlipped: boolean;
  onFlip: () => void;
}
```
